# 列表

## 列表定义

列表是Python中最基本的数据结构。是一些元素（如字符串，数字，对象）的有序集合，用中括号包围，
每个元素用逗号分隔开，每一个元素都有其对应的位置，称为索引，

列表内的元素不必全是相同的类型。

## 列表创建

创建一个列表，只要把逗号分隔的不同的数据项使用方括号括起来即可。

```python
a = ['bob', 'sam', 1000, 1234]
```
